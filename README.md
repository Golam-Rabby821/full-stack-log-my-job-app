# LogMyJobs

Personal job application tracker built with React, Vite, and shadcn-ui.

## Local development

```sh
# install deps
npm install

# start dev server
npm run dev

# run lint
npm run lint

# build for production
npm run build
```

## Deployment

Deploy the built `dist` folder to your hosting of choice (Netlify, Vercel, Cloudflare Pages, etc.). Example:

```sh
npm run build
# then point your host to the dist/ folder
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
